# Produktfähigkeiten

## Task-Management *(FEAT-1, seit 2026-04-03)*

Der Nutzer kann Tasks mit einem Titel erfassen, bearbeiten, als erledigt markieren und löschen – alles auf einem einzigen Screen ohne Seitenreload. Erledigte Tasks bleiben sichtbar und sind visuell klar unterscheidbar (durchgestrichen, gedämpfte Farbe). Alle Tasks werden automatisch im Browser-Speicher (localStorage) persistiert und überleben einen Reload ohne Datenverlust.
